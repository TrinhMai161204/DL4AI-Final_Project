# DL4AI-Final_Project

This project uses LSTM neural networks for stock price prediction, trading signal identification, profitable stock selection, risk management, and portfolio composition using Nasdaq and Vietnam stock market datasets.

## Tasks
- Nasdaq forecasting
- Vietnam forecasting
- Trading signal classification
- Portfolio optimization

## Requirements
- Python
- TensorFlow
- Scikit-learn
- Pandas
- Matplotlib

## Running
Open the Jupyter notebook and run cells sequentially.

## Dataset

The datasets were too large to upload to GitHub.

Please place:
- data_nasdaq_csv.zip
- data-vn-20230228.zip

inside the project directory before running the notebook.
